SAM Fine-tuning for Medical Imaging

Genoray FlexLab Internship Project

Computer Vision Medical AI Fine-tuning LoRA

Authors

H. Choi (Joint First Author)
Y. Uhm (Joint First Author)

Abstract

This project focuses on fine-tuning the Segment Anything Model (SAM) for dental X-ray segmentation tasks. We employed LoRA (Low-Rank Adaptation) based tuning strategies to adapt the pre-trained SAM model for medical imaging applications, specifically targeting dental X-ray analysis.

Methodology

  • Base Model: Segment Anything Model (SAM)
  • Fine-tuning Strategy: LoRA-based parameter-efficient tuning
  • Dataset: Dental X-ray images
  • Task: Medical image segmentation

Key Results

  • Successfully adapted SAM for medical imaging domain
  • Improved segmentation performance on dental X-ray data
  • Parameter-efficient fine-tuning using LoRA
  • Joint first authorship with Yoonhee Uhm
